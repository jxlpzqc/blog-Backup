# 项目规划 #

**应用名：** 错题宝  
**应用包名：** com.iziqiu.correctbook  
**需求：**  
1. 通过拍照收集文字版或图片版错题  
2. 可添加错解，错因分析  
3. 可在线同步错题（可选）  
4. 实现QQ或微信分享错题（可选）  
5. 自动生成DOC/PDF文档，方便打印错题本

**开发人数：**1人   
**开发语言：** Java  
**开发周期：** 15天  
**备注：**UI不是关键，丑就抽一点，重要的是实现业务。
